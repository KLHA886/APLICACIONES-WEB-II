Descripción del taller

Este taller práctico consiste en el desarrollo de una calculadora científica en consola (CLI) utilizando el lenguaje de programación Go. Permite realizar operaciones básicas (suma, resta, multiplicación y división) y operaciones más avanzadas como potencia y factorial, incluyendo validación de errores y un historial de operaciones.

El objetivo es aplicar conceptos fundamentales como variables, tipos de datos, estructura de control, bucles y entrada/salida por consola.
